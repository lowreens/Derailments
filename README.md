Derailments
===========
Statistical analysis of derailments between 2008 and 2013 Q2 in Lithuania.  

Variables
---------

1. Rolling stock
  * model
2. Driver
  * License issue date
  * Allowed opperational area
  * Sober
  * Resting time
3. Maneuver coordinator
  * License issue date
  * Allowed opperational area
  * Sober
  * Resting time
4. Enviroinment
  * Weather
  * Time
5. Infrastructure
  * Rail type
  * Derailment place
  * Work load of a shift
